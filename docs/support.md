# サポート

KintaiFlowのご利用に関するご質問やご相談は、以下の方法でサポートを受けられます。

## お問い合わせ方法

- サポート専用フォーム（準備中）
- メール: support@kintaiflow.example.com
- よくあるご質問は[FAQ](/docs/faq)ページもご参照ください

## サポート対応時間

- 平日 9:00〜18:00（祝日・年末年始を除く）

## サポート内容

- システムの操作方法やトラブル対応
- 機能追加・改善のご要望受付
- その他ご不明点へのご案内
