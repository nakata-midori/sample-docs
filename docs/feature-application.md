# 申請管理

有給休暇・残業・遅刻・早退などの各種申請をWeb上で簡単に行えます。

## 主な特徴
- 申請フォームは自動入力補助付き
- 承認フローのカスタマイズが可能
- 申請・承認状況の通知機能
- 申請履歴の検索・ダウンロード

## 詳細な使い方
1. 「申請」メニューから申請種別を選択
2. 必要事項を入力し、送信
3. 承認状況はマイページで確認可能

## 管理者向け機能
- 申請内容の一括承認/却下
- 申請履歴のエクスポート
